# Create Your Library - Backend

**Created by Noah Murphy**

**API:** This project serves as the entry point of the API application.
It contains controllers, models, and routes to handle incoming HTTP requests.

**infrastructure:** This project is responsible for managing data storage and access to such.
It uses a MongoDB as the database.
